# 05. 操作・演出・サウンド仕様 (Interaction, Animation & Sound)

プレイヤーの操作方法、正誤判定時の人形のリアクション演出、およびゲームを彩るサウンド（SE/BGM）に関する仕様です。

---

## 1. 操作方法
ユーザー環境に合わせて、直感的かつ遅延のない操作を提供します。
- **PC環境**:
  - すべてのアクション（スタート、診療科選択、リザルト確認など）は **マウスのクリック操作** で行います。
  - 診療科選択ボタンは、ホバー時（カーソルを乗せたとき）に少し大きくなる、または色が変わるなどのフィードバックを表示します。
- **スマートフォン・タブレット環境**:
  - すべてのアクションは **画面のタップ（タッチ）操作** で行います。
  - 誤操作（ダブルタップによるズームなど）を防ぐため、適切なViewport設定やCSS（`touch-action: manipulation`）を適用します。

## 2. 人型アバター（人形）のリアクション演出
画面中央に表示されるシンプルな人形は、プレイヤーの選択結果に応じて以下のように感情豊かなアニメーションを行います。

- **正解時 (嬉しそうなリアクション)**:
  - **ビジュアル変化**: 人形が「笑顔（目を細めるなど）」になり、体全体が上方向にピョンピョンと弾む（ジャンプする）ような嬉しい動きをします。さらに、両腕は万歳をするように上向きに大きく回転します（腕と袖がバラバラにならないよう、グループ化して肩を支点に一体としてアニメーションします）。
  - **エフェクト**: 人形の周囲にキラキラした星やハートなどのポジティブなエフェクトが一時的に発生します。
- **不正解・時間切れ時 (ガッカリしたリアクション)**:
  - **ビジュアル変化**: 人形が「がっかりした表情（落胆、目をバツ印にするなど）」になり、肩を落としたり、左右にがっくりと揺れたり（うなだれる動き）をします。さらに、両腕はだらりと下向きに垂れ下がる、または小刻みに震えるように回転します（腕と袖がグループ化され、肩を支点に一体としてアニメーションします）。

## 3. サウンド仕様
効果的なサウンド演出により、ゲームの没入感とテンポ感を高めます。

### 3.1. 効果音 (SE)
正誤判定時に、指定された音声ファイル（MP3形式）を再生します。
- **正解時**:
  - ファイルパス: `/public/seikai.mp3`
  - 再生タイミング: 正解ボタンを押した、または正解の判定が出た瞬間に1回再生。
- **不正解・時間切れ時**:
  - ファイルパス: `/public/matigai.mp3`
  - 再生タイミング: 誤ったボタンを押した、または1問の制限時間が切れてタイムアップになった瞬間に1回再生。

### 3.2. BGM (バックグラウンドミュージック)
ゲームプレイを盛り上げる、テンポが良くて聞き飽きない「いい感じのオリジナルBGM」を設定します。
- **BGMの仕様**:
  - ファイルパス: `/public/bgm.mp3`
  - **曲調**: 医療や仕分けのスピード感を意識した、明るく軽快でコミカルなテクノポップ、またはアップテンポなシンセサイザー音源。
  - **再生タイミング**: ゲーム画面（クイズ中）に入った瞬間に自動でループ再生を開始し、ゲームオーバーまたはタイムアップ（90秒終了）でフェードアウトします。
  - **ループ処理**: 曲の終端に達したら、継ぎ目なくシームレスに最初からループ再生されるようにします。

### 3.3. 音声ファイル欠損時の代替処理 (Web Audio API フォールバック)
実装するAIモデルがアセット（MP3ファイル）が見つからない環境でビルドした場合でもエラー落ちしないよう、以下の代替処理（フォールバック）を実装要件に含めます。
- **仕様**: 音声ファイル（`/public/seikai.mp3`、`/public/matigai.mp3`、`/public/bgm.mp3`）の読み込みに失敗した場合、ブラウザの **Web Audio API** を使用して、コード上で以下のような音を動的に生成して鳴らすこと。
  - **正解音 (seikai.mp3 代替)**: 「ピピーン！」と高音のサイン波（例：1000Hz ➔ 1200Hz）を短時間で2音鳴らす。
  - **不正解音 (matigai.mp3 代替)**: 「ブブー」と低音のノコギリ波（例：150Hz ➔ 100Hz）を少し濁らせて鳴らす。
  - **BGM (bgm.mp3 代替)**: Web Audio API のオシレーターとシーケンサーを用いて、簡単な8ビート調の電子音メロディ（ドレミソ...などの短いループ）をテンポに合わせて自動生成しバックグラウンドでループ演奏する。
