# 03. 画面レイアウトとUI演出 (UI & Layout)

プレイヤーが直感的に状況を把握し、素早く回答できるための画面構成とビジュアル表現の仕様です。

---

## 1. メインゲーム画面レイアウト
メイン画面は、情報を一目で整理できるよう、上部・中央・下部の3セクション構成です。

```
+-------------------------------------------------------------+
| [全体時間: 154s]      [スコア: 120]      [残機: ◆◆◆◇]     |
+-------------------------------------------------------------+
|                                                             |
|                       ▲  (頭)  [矢印 ↙]                     |
|                     / | \                                   |
|                    /  |  \ (体イラスト)                     |
|                      / \                                    |
|                     /   \                                   |
|                                                             |
|   ------------------[ 患者の詳細情報 ]------------------   |
|   対象部位: 「頭」                                          |
|   詳細症状: 「頭がズキズキと脈打つように痛い」              |
|   ------------------------------------------------------    |
|                                                             |
|       +---------------------------------------------+       |
|       | [1問タイマー: 8.5s] (バー表示)              |       |
|       +---------------------------------------------+       |
+-------------------------------------------------------------+
|  [ 内科 ]  [ 整形外科 ]  [ 耳鼻咽喉科 ]  [ 皮膚科 ]  [ 眼科 ]   |
+-------------------------------------------------------------+
```

### 1.1. ステータスエリア (画面上部)
- **全体残り時間**: `180秒` からのカウントダウン数値。
- **現在のスコア**: デジタルフォントや視認性の高いUIで表示（1問正解ごとに10点加算）。
- **残機表示**: ハートマークやライフゲージ（◆◆◆◆ ➔ 最大4つ）で視認性を確保。

### 1.2. クイズステージエリア (画面中央)
- **患者キャラクター（イラスト）**:
  - 画面中央に、**白や水色を基調とした、清潔感のあるシンプルな「ロボット風アバター」**のイラストを配置します。
  - 特定の個人の容姿に依存しないニュートラルなデザインとし、ロボットの身体の各部位（頭、目、耳、胸、お腹、膝、手足など）に矢印やマーカーを分かりやすく指し示せる構造にします。
- **部位へのダイレクト矢印表示**:
  - イラスト内の特定の患部（頭、目、耳、鼻、皮膚、腕、膝、お腹など）に対して、直接赤い「矢印」を指し示します。
- **詳細情報表示（イラスト下部）**:
  - イラストのすぐ下に、すっきりとした枠で囲まれた詳細情報を表示します。
    - **対象部位**: 例：「頭」「膝」「目」
    - **詳細症状（症状テキスト）**: 例：「ズキズキと脈打つように痛い」「曲げると激痛が走る」
- **1問あたりの制限時間インジケーター**:
  - 現在のフェーズ制限時間（10秒/8秒/6秒）の経過を表すプログレスバー。時間が減るにつれて緑 ➔ 黄 ➔ 赤へ変化。

### 1.3. 回答アクションエリア (画面下部)
- **診療科ボタン（5種類固定表示）**:
  - **「内科」「整形外科」「耳鼻咽喉科」「皮膚科」「眼科」** の5つのボタンを横一列、または整然としたグリッド状に配置。
  - タップ・クリックしやすいよう、十分な大きさ（44px以上）を確保。
  - 回答ボタンをタップした瞬間に、ボタンが沈み込むようなフィードバック演出を返します。

## 2. 正誤判定エフェクト
- **正解時**:
  - 画面に大きく「〇（正解）」を表示し、小気味よい効果音を再生。
  - 患者キャラクターが「ホッとした表情」や「お辞儀」をするなどのアニメーション。
- **不正解・時間切れ時**:
  - 画面に大きく「✕（不正解 / タイムアップ）」を表示。
  - 残機が減るビジュアル演出（ハートが割れる等）。
