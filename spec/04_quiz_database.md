# 04. クイズデータと解説仕様 (Quiz Data & Explanation)

患者の症状（クイズ問題）のデータ構造、仕分け先（診療科）、およびゲーム終了後に表示する解説機能の仕様です。

---

## 1. クイズデータの基本構造
各問題は、以下のデータフィールドを持つオブジェクトとして定義します。

| フィールド名 | 型 | 説明 | 例 |
| :--- | :--- | :--- | :--- |
| `id` | string | 問題の一意の識別子 | `"q_001"` |
| `part` | string | 症状が発生している部位（矢印を指す場所） | `"お腹"` |
| `symptom` | string | 画面に表示する症状テキスト | `"数日前からみぞおちのあたりがキリキリ痛む"` |
| `answer` | string | 正解となる診療科（仕分け先） | `"内科"` |
| `explanation`| string | ゲーム終了時に表示する解説文 | `"みぞおちや胃の痛み、消化不良などは「内科」で診察します。"` |

## 2. 診療科の仕分けルールと具体例
ゲームを面白く、また学びになるように、以下5つの診療科の仕分け例を定義します。

### 内科 (Internal Medicine)
- 症状が発生する部位：お腹、頭、胸 など
- 症状例：「風邪っぽい、熱がある」「お腹を下している」「頭が重い、だるい」
- 解説例：「熱や風邪、胃腸の不調などは身体の内側の病気を扱う『内科』が適しています。」

### 整形外科 (Orthopedics)
- 症状が発生する部位：膝、腰、肩、腕、足首 など
- 症状例：「転んで足をくじいた」「階段の上り下りで膝が痛い」「腰が痛くて曲げられない」
- 解説例：「骨や関節、筋肉、靭帯などの痛みやケガは『整形外科』が専門です。」

### 耳鼻咽喉科 (Otolaryngology)
- 症状が発生する部位：耳、鼻、喉（のど）
- 症状例：「耳の奥が詰まった感じがする」「鼻水とくしゃみが止まらない」「喉がヒリヒリして痛い」
- 解説例：「耳、鼻、喉（のど）に起こるトラブルは『耳鼻咽喉科』で治療を行います。」

### 皮膚科 (Dermatology)
- 症状が発生する部位：皮膚（手、腕、顔、背中、足など）
- 症状例：「腕に赤いブツブツ（湿疹）ができてかゆい」「日焼けをしすぎて肌がヒリヒリ痛む」
- 解説例：「湿疹、じんましん、かぶれなど、皮膚（肌）に現れる症状は『皮膚科』を受診します。」

### 眼科 (Ophthalmology)
- 症状が発生する部位：目
- 症状例：「目がゴロゴロして異物感がある」「最近急に視界がかすんで見えにくい」「結膜が赤く充血している」
- 解説例：「目の痛み、かゆみ、見えにくさなど、目に関わる不調は『眼科』が専門です。」

## 3. ゲーム終了時の解説機能
プレイヤーが間違えた問題に対する復習をサポートするため、ゲーム終了時の解説画面を構築します。

### 仕様詳細:
- **記録タイミング**:
  - プレイヤーが「不正解のボタンを押した」または「現在の問題制限時間が経過（タイムアップ）」した問題を、ゲーム進行中に一時的な配列（例：`wrongAnswers`）へ保存します。
- **表示タイミング**:
  - ゲーム終了時（180秒経過、または残機0時）に、スコア表示の下部、あるいは「解説モード」として一覧表示します。
- **UI表現**:
  - 間違えた問題ごとに以下を表示：
    - 「患者の症状（部位＋テキスト）」
    - 「あなたの回答（またはタイムアップ）」 ➔ 赤字
    - 「正しい診療科」 ➔ 青字
    - 「詳しい解説（解説文）」 ➔ 分かりやすい説明テキスト
  - **縦スクロール対応（Scroll Area）**:
    - 間違えた問題がたくさんある場合（例：3問以上など）、リザルト画面全体のレイアウトが崩れたり、画面外にボタンが押し出されたりするのを防ぐため、解説リストエリアに最大高さ（Max Height）を設定します。
    - 解説リストエリア内を**「縦スクロール（垂直スクロール）」**させることで、プレイヤーが画面を下にスムーズにスクロールしながら、すべての解説を一覧で確認できるようにします。
    - スクロールバーはスクロール可能であることが視覚的に分かりやすいデザインにします。
- **クイズ問題の総数**:
  - 本作は、合計 **30問** のクイズデータをデータベースとして保持します。
  - ゲーム中、これら30問の中からランダムに患者（問題）が出現します。
  - 1プレイ（180秒間）中に同じ問題が何度も重複して出現しないよう、一度出題された問題はプールから除外され、全問消化した後に再度リセットされる仕組み（シャッフル・ディール方式）を推奨します。

## 4. クイズ問題30選リスト (30 Questions List)

実際にゲームに組み込む30問のデータ定義です。

### ■ 内科 (6問)
1. **部位: お腹**
   - 症状: 「数日前からみぞおちのあたりがキリキリ痛む」
   - 解説: みぞおちや胃の痛み、消化不良などは身体の内側の臓器を扱う「内科（消化器内科）」が適しています。
2. **部位: 胸**
   - 症状: 「動悸がして、胸のあたりが締め付けられるように息苦しい」
   - 解説: 心臓や血管、呼吸器系の異常など、胸の痛みや息苦しさは「内科（循環器・呼吸器内科）」を受診します。
3. **部位: 頭**
   - 症状: 「風邪をひいたみたいで、熱があって体がゾクゾク寒気がする」
   - 解説: 発熱、悪寒、だるさなど全身性の感染症（風邪やインフルエンザなど）は「内科」の領域です。
4. **部位: お腹**
   - 症状: 「冷たいものを食べすぎたせいか、下痢と腹痛が止まらない」
   - 解説: 急性の胃腸炎や消化不良による下痢・腹痛は「内科」で診察・投薬を行います。
5. **部位: 全身**
   - 症状: 「最近、やたらと喉が渇いて尿の回数が増え、体がだるい」
   - 解説: 糖尿病などの代謝異常、生活習慣病の疑いがある全身症状は「内科（糖尿病・内分泌内科）」の受診が推奨されます。
6. **部位: 胸**
   - 症状: 「コンコンと激しい咳が数週間続いていて、夜も眠れない」
   - 解説: 長引く咳や気管支の不調など、呼吸器系のトラブルは「内科（呼吸器内科）」が専門です。

### ■ 整形外科 (6問)
7. **部位: 膝**
   - 症状: 「階段を下りるときに、膝の関節がきしんで激しく痛む」
   - 解説: 骨や関節、軟骨のすり減りによる関節痛は「整形外科」が専門です。
8. **部位: 腰**
   - 症状: 「重い荷物を持ち上げた瞬間に、腰にギクッと激痛が走った」
   - 解説: いわゆる「ぎっくり腰（急性腰痛症）」や腰椎のトラブルは「整形外科」を受診します。
9. **部位: 足首**
   - 症状: 「段差を踏み外して足首を強くひねり、青く腫れて歩けない」
   - 解説: 骨折や捻挫（靭帯の損傷）、打撲などの外傷は「整形外科」の領域です。
10. **部位: 肩**
    - 症状: 「腕を上に上げようとすると、肩に引っかかるような痛みがあって上がらない」
    - 解説: 五十肩（肩関節周囲炎）や腱板の損傷など、肩関節 of 運動障害は「整形外科」が適しています。
11. **部位: 首**
    - 症状: 「朝起きたら首が固まっていて、どちらを向いても激痛が走る」
    - 解説: 寝違えや頚椎（首の骨・神経）の異常による急な痛みは「整形外科」で診察します。
12. **部位: 手首**
    - 症状: 「スマホの使いすぎか、親指の付け根の手首が腫れて動かすと痛い」
    - 解説: 腱鞘炎（けんしょうえん）など、使いすぎによる腱や筋肉の炎症は「整形外科」の専門領域です。

### ■ 耳鼻咽喉科 (6問)
13. **部位: 耳**
    - 症状: 「耳の奥が詰まった感じがして、キーンという金属音が聞こえる」
    - 解説: 耳鳴りや耳閉感（耳が詰まった感じ）は、耳の奥の神経や鼓膜を扱う「耳鼻咽喉科」を受診します。
14. **部位: 鼻**
    - 症状: 「花粉の季節になると、鼻水とくしゃみが止まらず鼻が完全に詰まる」
    - 解説: アレルギー性鼻炎や副鼻腔炎（ちくのう症）など、鼻の粘膜トラブルは「耳鼻咽喉科」が専門です。
15. **部位: 喉（のど）**
    - 症状: 「つばを飲み込むだけでも、喉の奥がヒリヒリして焼けるように痛い」
    - 解説: 扁桃腺（へんとうせん）の腫れや喉の炎症は「耳鼻咽喉科」で直接処置を受けるのが効果的です。
16. **部位: 耳**
    - 症状: 「プールに入った後から、耳の奥がかゆくてズキズキ痛むようになった」
    - 解説: 外耳道炎や中耳炎など、耳の内部への細菌感染による痛みは「耳鼻咽喉科」が専門です。
17. **部位: 鼻**
    - 症状: 「急に鼻血が出てしまい、小鼻を圧迫してもなかなか止まらない」
    - 解説: 鼻の粘膜（キーゼルバッハ部位など）からの出血や処置は「耳鼻咽喉科」が適しています。
18. **部位: 頭（耳・平衡感覚）**
    - 症状: 「急に目の前がグルグル回り、まっすぐ歩けなくなるめまいがする」
    - 解説: 内耳（三半規管）の異常からくる回転性のめまいは、耳の専門である「耳鼻咽喉科」を受診します。

### ■ 皮膚科 (6問)
19. **部位: 腕**
    - 症状: 「腕に赤いブツブツとした湿疹ができて、かきむしるほどかゆい」
    - 解説: 湿疹、じんましん、アトピー、かぶれなど、皮膚表面のあらゆる炎症は「皮膚科」が専門です。
20. **部位: 顔**
    - 症状: 「ニキビが化膿して真っ赤に腫れ上がり、痛みを伴うようになってきた」
    - 解説: 尋常性痤瘡（ニキビ）は皮膚の毛穴の感染症であり、悪化させないためにも「皮膚科」で治療します。
21. **部位: 足**
    - 症状: 「足の指の間がじゅくじゅくして皮がむけ、強いかゆみがある」
    - 解説: 白癬菌というカビの一種による感染症である「水虫」は、「皮膚科」で顕微鏡検査と適切な軟膏処方を行います。
22. **部位: 背中**
    - 症状: 「背中の片側に沿ってピリピリとした痛みが走り、小さな水ぶくれが集まってできた」
    - 解説: 免疫低下時に起こる「帯状疱疹（たいじょうほうしん）」は、早急に「皮膚科」で抗ウイルス薬を処方してもらう必要があります。
23. **部位: 手**
    - 症状: 「新しい洗剤を使い始めてから、手のひらがカサカサに荒れてひび割れて痛い」
    - 解説: 接触皮膚炎（手荒れ・主婦湿疹）など、特定の物質に対する皮膚の防衛反応は「皮膚科」の診断が必要です。
24. **部位: 足**
    - 症状: 「足の裏に硬い芯のような突起ができて、歩くたびにピンポイントで痛い」
    - 解説: 魚の目（鶏眼）やタコ、ウイルス性のイボなどは「皮膚科」で芯の除去や窒素治療を行います。

### ■ 眼科 (6問)
25. **部位: 目**
    - 症状: 「朝起きたら目やにで目が開かず、白目が真っ赤に充血している」
    - 解説: 結膜炎（細菌性やウイルス性、はやり目など）による充血や目やには「眼科」が専門です。
26. **部位: 目**
    - 症状: 「まぶたの一部が赤くプツッと腫れて、まばたきをするとズキズキ痛い」
    - 解説: 麦粒腫（ものもらい・めばちこ）はまぶたの分泌腺の細菌感染であり、「眼科」で点眼薬などの治療を行います。
27. **部位: 目**
    - 症状: 「最近、急に視界全体が霧がかかったようにかすんで、文字が見えにくい」
    - 解説: 白内障や緑内障、視力低下など、眼球内部のピント調整や視神経の不調は「眼科」の受診が必要です。
28. **部位: 目**
    - 症状: 「目に何かゴミが入ったようで、ゴロゴロして涙が止まらず開けられない」
    - 解説: 角膜（目の表面）の傷や異物混入は、眼球を傷つけないよう「眼科」で安全に除去・処置します。
29. **部位: 目**
    - 症状: 「夕方になると目が異常に乾いてシパシパし、奥が重く疲れる」
    - 解説: ドライアイや眼精疲労、涙の分泌異常などは「眼科」で適切な目薬の処方を受けます。
30. **部位: 目**
    - 症状: 「目の前を小さな虫やゴミのような黒い影がフワフワ飛び回って見える」
    - 解説: 飛蚊症（ひぶんしょう）と呼ばれる症状で、網膜剥離などの重大な病気の前兆である可能性もあるため「眼科」で精密検査を行います。
